# Luxury Chocolate — Šokolādes ar logo uzņēmumiem

Premium korporatīvo šokolādes dāvanu mājaslapa. Personalizētas šokolādes ar uzņēmuma logo klientiem, partneriem, darbiniekiem un pasākumiem.

🌐 **Mājaslapa**: [luxurychocolate.lovable.app](https://luxurychocolate.lovable.app)

## Iespējas

- 🍫 9 SEO optimizētas mērķlapas (LV)
- 🇬🇧 Angļu un 🇷🇺 krievu valodas versijas (27 lapas kopā)
- 📊 JSON-LD strukturētie dati (Product, FAQPage, Article, BreadcrumbList)
- ⚡ Lazy loading & code splitting
- 📱 Pilnībā responsīvs dizains
- 🎬 Video hero sekcija
- 📝 Piedāvājuma pieprasījuma forma ar e-pasta paziņojumiem

## Tehnoloģijas

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) — ātra izstrāde un build
- [Tailwind CSS](https://tailwindcss.com/) — utility-first stili
- [shadcn/ui](https://ui.shadcn.com/) — UI komponenti
- [Framer Motion](https://www.framer.com/motion/) — animācijas
- Lovable Cloud — backend, e-pasts, edge functions

## Izstrāde

```sh
git clone https://github.com/ilzeeisaka-alt/luxurychocolate.git
cd luxurychocolate
npm i
npm run dev
```
